A scalable data ingestion endpoint deployed on AWS using CloudFormation with
automated CI/CD and devcontainer development environment.

## Overview

This project provides a data ingestion API endpoint hosted on AWS infrastructure
with Infrastructure as Code using CloudFormation and automated continuous
integration via GitHub Actions.

## Features

- Data ingestion API endpoint
- AWS-hosted infrastructure
- CloudFormation Infrastructure as Code
- Automated CI/CD pipeline
- Devcontainer development environment
- Pre-commit hooks
